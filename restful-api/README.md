# RESTful API

## Description
This project is an introduction to RESTful APIs. It covers:
- HTTP/HTTPS basics
- API consumption with command line (curl)
- API consumption with Python (requests)
- API development with http.server and Flask

## Files
* `task_02_requests.py`: A Python script that fetches posts from JSONPlaceholder, prints the titles, and saves the data into a `posts.csv` file.
